## Introduction 


This is my healthcare data analysis project. This project is created using the kaggle healthcare dataset ([https://www.kaggle.com/datasets/austinreese/craigslist-carstrucks-data](https://www.kaggle.com/datasets/prasad22/healthcare-dataset)).

### Key Features 

1. 'Name': = The name of the patient

2. 'age': The age of the patient

3. 'gender': The gender of the patient

4. 'blood_type': The Blood type of the patient

5. 'medical_condition': The Medical condition of the patient

6. 'date_of_admission': The date of admission of the patient

7. 'Doctor: Provides name of the doctor

9. 'Hospital': Name of the hospital

10. 'Insurance": Name of the the insurance

11. 'billing_amount": Describes the bill of of the patient

12. 'Room Number': Room number of the patient.

13. 'Admission type': Describes the situation of admission of the patient

14. 'discharge date': The discharge date of the patient

15. 'Medication': Medication of the patient

16. 'Test Results': Test results of the patient
